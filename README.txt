# MedBot-
Step 1:Open cmd and direct to your project folder
Step 2:Connect the SQL server connection from VsCode
Step 3:Start the php server by typing the php -S command with localhost:8000 address
Step 5:Open your browser(Chrome or Edge) and type the address given in the cmd
Step 6:The Authentication page is opened
Step 7:Enter your details and signin if new user signup and then login
Step 8:MedBot interface appears and then you start to enter the symptoms and get predictions based on your symptoms
